Fine-Tuning

Fine-tuning means taking a model that already learned general patterns and training it a little more for a specific task or style. You don't start from scratch, you build on what the model already knows.

Pretraining vs Fine-Tuning

Broad learning, then focused learning

Pretraining builds a general base model. Fine-tuning shapes that base model into a specialist.

Pretraining
Learns broad patterns from large data.
DataHuge and general
ResultA base model
Fine-Tuning
Learns a specific task from focused data.
DataSmall and focused
ResultA specialized model

Key takeaways
  • • Fine-tuning starts from a pre-trained model.
  • • It adapts the model for a specific task or style.
  • • It usually uses focused examples.
  • • It is different from training a model from scratch.
  • • Fine-tuning is useful when consistent behavior is needed.
