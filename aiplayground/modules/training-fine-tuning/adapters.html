Adapters

Adapters are small trainable modules added to a model. They help specialize the model for a task without changing the full base model.

Why Adapters Help

Full fine-tuning vs adapters

Both can specialize a model. Adapters just do it in a lighter, easier-to-manage way.

Full Fine-Tuning
Changes many parts of the model
Needs more storage for each specialized model
One full copy per task
Adapters
Keep the base model mostly unchanged
Store small task-specific modules
Easier to switch between tasks
Adapter Switching

One base model, many adapters

Instead of keeping many full models, we can keep one base model and attach different adapters for different needs. Click a branch to switch.

How It Fits Together

Three simple steps

1
Start with one base model that stays the same.
2
Train a small adapter for each task.
3
Attach or swap adapters to change behavior.

Key Takeaways

  • • Adapters are small add-on modules.
  • • They specialize a base model for a task.
  • • The base model can stay mostly unchanged.
  • • Different adapters can be swapped for different use cases.
  • • Adapters make customization easier to manage.
