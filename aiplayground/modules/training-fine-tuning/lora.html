LoRA

LoRA is a fine-tuning method that keeps the main model mostly unchanged and trains a small set of extra weights. This makes fine-tuning faster and cheaper.

Why LoRA Helps

A simple way to think about it

Instead of rewriting an entire book, LoRA adds small notes that guide how the book should be read.

The base model stays the same, but the adapter changes how the model behaves for a specific task.

Key takeaways
  • • LoRA is a lightweight fine-tuning method.
  • • It avoids updating the full model.
  • • It trains small adapter weights.
  • • The base model can stay mostly unchanged.
  • • LoRA makes fine-tuning more efficient and easier to manage.
