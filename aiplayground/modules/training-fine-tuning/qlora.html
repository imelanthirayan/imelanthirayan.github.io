QLoRA

QLoRA is like LoRA, but with an extra memory-saving step. It uses a smaller version of the base model in memory while training small LoRA adapters.

Why Quantization Helps

A simple way to think about it

Imagine carrying a full-size photo versus a compressed photo. The compressed one uses less space, but still keeps enough useful detail.

Quantization reduces how much memory the model needs while still keeping it useful for fine-tuning.

Key takeaways
  • • QLoRA builds on LoRA.
  • • LoRA trains small adapters.

  • • Quantization helps large models fit into smaller hardware.
  • • QLoRA is useful when memory is limited.
