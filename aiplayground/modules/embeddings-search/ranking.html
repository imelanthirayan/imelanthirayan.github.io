🎓 Tips 2 tips
A high score doesn't guarantee relevance, set a minimum threshold to filter out low-scoring noise.
Reranking is a second step, retrieve 100 results fast, then run a slower but more accurate reranker on just those 100.

Key Takeaway

AI usually finds many possible results, then ranks them so the most relevant ones appear first.
