🎓 Tips 3 tips
Filter first, then search. Narrowing by category or year shrinks the search space before matching happens.
Filters can hide the best answer. If you filter too strictly, the right document may get excluded entirely, try clearing a filter to see.
This demo uses simple keyword overlap for search. Real RAG systems combine metadata filters with vector similarity search.
💡
Key Takeaway

Metadata filtering helps RAG search the right subset of documents before looking for the best answer.
