What is Model Optimization?

After a model has been trained, it can be optimized to make it faster, use less memory, or run on smaller devices.

Where It Happens

Optimization comes after training

1
Train the model so it learns the task.
2
Optimize it for speed, memory, or size.
3
Deploy it where it needs to run.

Key Takeaways

  • • Optimization happens after training.
  • • Different optimizations have different goals.
  • • Some reduce memory.
  • • Some improve speed.
  • • Some involve small trade-offs in accuracy.
  • • The right choice depends on where the model will run.

Summary

Model optimization helps AI models become faster, smaller, and more efficient. Different techniques are used depending on the needs of the application.
