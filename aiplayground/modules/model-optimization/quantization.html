Quantization

Quantization changes how a trained model stores its numbers. The goal is to make the model smaller and faster — without changing what it has learned.

Key Takeaways

  • • Quantization changes how numbers are stored.
  • • The model knowledge stays the same — it is not retrained.
  • • Quantization reduces memory usage.
  • • It often improves inference speed.
  • • Lower precision may slightly reduce accuracy.

Summary

Quantization makes trained AI models smaller and faster by storing numbers more efficiently. It is one of the most common optimization techniques used during deployment.
