Token Cost Estimator

AI services often count text using tokens. More tokens usually means more cost.

This is a simple learning estimator, not exact pricing for any real AI provider.

Interactive Playground
Cost Formula
Cost = (input tokens × input rate)
+ (output tokens × output rate)
Example (GPT-4o mini)
📥 100 input tokens × $0.15 / 1M = $0.000015
📤 50 output tokens × $0.60 / 1M = $0.000030
💰 Total $0.000045

💡 Output tokens are always more expensive than input tokens, AI generating text costs more than reading it.
